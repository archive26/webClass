float

블록 태그들을 나란히 배치하는데 목적이 있다

둥~~~~ 띄워서 부모의 컨텐트박스안에서 왼쪽 혹은 오른쪽으로 배치하는 방법(left/right)

둥~~~~뜰 때 문서의 흐름이 제외된다 -> (부모태그입장에서 집나간 자식으로 간주된다.)

clear

float를 해제할 때 사용하는 속성 - clear속성이 적용된 태그는 앞서 만들어진 형제태그 중에서 float가 되어있는 태그 중 가장 아래까지 내려와있는 floating태그 바로 밑에 배치가된다.이 때, clear속성이 지정된 태그가 배치가 되면서 부모태그는 height의 지정값이 auto이기 때문에 clear속성이 지정된 태그가 있는 곳까지 높이를 자동으로 늘린다
